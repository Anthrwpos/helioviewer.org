Method Detail
buildMovie(self, layers)
Uses the layers passed in to send an Ajax request to api.php, to have it build a movie. Upon completion, it displays a notification that lets the user click to view it in Shadowbox or download the high quality version.
Parameters:
{Object} self
{Object} layers
-- An array of layer strings in the format: "obs,inst,det,meas,visible, opacity'x'XStart,XEnd,YStart,YEnd,offsetX,offsetY"
checkMovieLayers(self, visibleCoords)
Checks to make sure there are 3 or less layers in the movie. If there are more, user is presented with pop-up in Shadowbox asking them to pick 3 layers.
Parameters:
{Object} self
{Object} visibleCoords
-- An array containing the top, left, bottom, right coordinates of: 1) what is currently visible in the viewport, or 2) a selected region within the viewport. Note that these coordinates are heliocentric, with the center of the sun being (0,0).
playMovie(helioviewer, imgWidth, imgHeight, data)
Calculates what dimensions the image has, whether to scale it or not, and how big Shadowbox should be. Opens Shadowbox to play the movie.
Parameters:
helioviewer
imgWidth
imgHeight
data
updateProgress(timeout)
Displays a percentage, starting at 0, and increments it by one every seconds. It is here so users can tell something is going on.
Parameters:
{Object} timeout
-- a value in miliseconds that was calculated in buildMovie as an estimate of how long the movie will take to build / 100.
